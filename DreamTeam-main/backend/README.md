# Dream Team Game Server

Серверная часть для браузерной игры в жанре «Бункер» в IT-тематике.  
На данный момент реализована базовая функциональность лобби.

## Что реализовано

- **`POST /api/room/create`** — создание новой игровой комнаты.  
  Генерируется уникальный код комнаты, создаётся игрок-создатель, устанавливается httpOnly cookie с идентификатором игрока.

- **`POST /api/room/join`** — присоединение к существующей комнате.  
  По коду комнаты добавляется новый игрок, ему также выдаётся httpOnly cookie.

## Тестирование

Для проверки работы эндпоинтов используется временный HTML-файл (`public/index.html`), который позволяет создавать комнаты, присоединяться к ним и отправлять тестовый авторизованный запрос.

## Стек

- Node.js
- Express
- SQLite (через собственный класс `DataStorage`)
- cookie-parser, cors

## Запуск

```bash
npm install
node server.js
```

Сервер будет доступен по адресу `http://localhost:3000`.  
Статический тестовый клиент открывается в браузере по тому же адресу.
